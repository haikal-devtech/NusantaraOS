# NusantaraOS 🇮🇩

Indonesia's first Linux distribution — for everyone.

> "Dari kepulauan, untuk semua."

## About
NusantaraOS is an Indonesian-first Linux distribution built on Arch Linux.
Automated, gaming-ready, and fully localized in Bahasa Indonesia.

## Status
🚧 Active development — Phase 1

## Roadmap
- Phase 1 — Arch Base Setup & Btrfs Layout
- Phase 2 — Wayland + KDE Plasma + PipeWire
- Phase 3 — Guardian Daemon + Sehat Check
- Phase 4 — Gaming Layer + Steam + Proton-GE
- Phase 5 — Calamares Installer + Zero-Panic Boot
- Phase 6 — v1.0 Public Release (Jawa)
